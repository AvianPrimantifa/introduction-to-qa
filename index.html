EMPLOYEE TOOLS

There are two important online tools for ICEHOUSE employee, which means, for us too, they are:
  • HARVEST

  • BAMBOO HR

WORK TOOLS

There are two important tools for ICEHOUSE Quality Analyst, they are:
  • JIRA
    : Flexible software for issue and project tracking. All stories, bugs, and
  • TEST RAIL
    : Web-based test case management software to manage, track and organize software testing efforts.

ADDITIONAL QA TOOLS

As a QA, we have few additional tools, mostly purpose for automation testing, such as:
  • JMETER
    : Graphical server performance testing tool, for both static and dynamic resources (files or CGI, Servlets, Perl scripts).
  • SELENIUM
    : Portable software testing framework for web applications.
  • INTELLIJ IDEA
    : Project text editor with Java integrated development environment (IDE) for developing computer software.
  • APPIUM
    : Open source test automation framework for use with native, hybrid and mobile web apps.
